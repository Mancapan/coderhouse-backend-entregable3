
- ============================================================
- ### USO DE LA COLECCIÓN DE POSTMAN ###
- Archivo: Entrega3-mongoDB-crud.postman_collection.json
- ============================================================
-
- Este proyecto incluye una colección de Postman que contiene
- todos los endpoints necesarios para probar la API (Products y Carts).
-

- ***
- ### PASO 1 – Ejecutar el servidor
- ***
- Antes de importar la colección, asegurarse de que el servidor
- esté en ejecución.
-
- ***
- ### PASO 2 – Importar la colección en Postman
- ***
- 1.  Abrir Postman.
- 2.  Hacer clic en "Import".
- 3.  Seleccionar "Upload Files".
- 4.  Buscar el archivo:
-
-      Entrega3-mongoDB-crud.postman_collection.json
-
- 5.  Hacer clic en "Import".
-
- La colección aparecerá en el panel izquierdo.
-
- ***
- ### PASO 3 – Probar los endpoints
- ***
- 1.  Expandir la colección.
- 2.  Seleccionar el endpoint deseado.
- 3.  Presionar "Send".
-
- Endpoints incluidos:
-
- PRODUCTS:
- - GET [/api/products](http://localhost:8080/api/products) => llamar a todos los productos con page y limit valores default.
- - GET /api/products/name/:title => Buscar productos por nombre
-
- CARTS:
- - GET /api/carts (http://localhost:8080/api/carts) => Buscar todos los carritos con sus productos en inventario
- - GET /api/carts/:cid => buscar carritos por id
- - POST /api/carts => Crear Carrito vacío
- - POST /api/carts/:cid/products/:pid => Agregar Producto a Carro
- - DELETE /api/carts/:cid/products/:pid => Elmiminar Producto de carro
- - DELETE /api/carts/:cid => Vaviar carrrito, Eliminar productos de carrito
- - PUT /api/carts/:cid => Actualizar productos de Carts
- - PUT /api/carts/:cid/products/:pid => Actualizar cantidad de UN producto en carts
-
- ***
- #### IMPORTANTE – Configuración de Base de Datos
- ***
- Para conectar con MongoDB Atlas para la revisión:
-
- 1.  Crear un archivo ".env" en la raíz del proyecto.
- 2.  Agregar la variable detallada en archivo = .env.example
